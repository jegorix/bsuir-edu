ЗАДАНИЕ

Лабораторная работа 7 посвящена конкурентному доступу к файлу с фиксированными записями с использованием `fcntl()` и блокировок чтения/записи.

СБОРКА

Перейти в каталог проекта:
`cd /Users/macbook/Documents/bsuir/osisp/lab-7-osisp`

Собрать debug:
`make`

Собрать release:
`make MODE=release`

Очистить сборку:
`make clean`

ЗАПУСК

Перейти в каталог нужной сборки:
`cd build/debug`

Создать файл с записями:
`./gen 10`

Запустить программу работы с записями:
`./records_work`

Для проверки конкурентного доступа откройте два терминала в одном и том же каталоге `build/debug` и запустите `./records_work` в обоих.

КОМАНДЫ

- `lst` — показать все записи
- `get N` — загрузить запись `N` (нумерация с 1)
- `mod` — изменить загруженную запись в рабочей области
- `put` — сохранить рабочую запись обратно в файл
- `inf` — показать рабочую, исходную и текущую файловую версии записи
- `q` — выйти

ПОВЕДЕНИЕ

- `GET` читает запись под разделяемой блокировкой
- `PUT` ставит эксклюзивную блокировку на запись, перечитывает её и сохраняет только если она не изменилась после последнего `GET`
- если запись успел изменить другой процесс, программа сообщает об этом, подгружает актуальную версию из файла и предлагает повторить модификацию
